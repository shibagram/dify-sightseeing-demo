# AI観光コンシェルジュ

日本各地の観光プラン・スポット・グルメを提案するAIチャットボットです。
[Dify](https://dify.ai/) のワークフロー（advanced-chat モード）として動作します。

## できること

- 旅行先のモデルコース提案（日程・予算・人数に応じて）
- 観光スポット・グルメ・イベント情報の案内
- 交通手段や移動ルートのアドバイス
- 画像のアップロードにも対応（JPG / PNG / GIF / WEBP、最大3枚）

## 対応言語

日本語と英語に対応しています。
チャット開始時に言語を選択でき、途中で切り替えることもできます。

英語モードでは、スポット名に日本語を併記し、費用を円と USD の両方で表示します。

## ワークフロー構成

```
開始 → 観光コンシェルジュ（LLM） → 回答
```

シンプルな3ノード構成です。会話履歴（直近20件）を保持するため、文脈を踏まえたやり取りができます。

## セットアップ

1. Dify の管理画面で「DSLファイルをインポート」を選択
2. `ai-tourism-concierge.yml` をアップロード
3. 「観光コンシェルジュ」ノードで使用するLLMモデルを設定
4. 公開して利用開始

## ファイル構成

```
ai-tourism-concierge.yml   # Dify DSL ファイル（ワークフロー定義）
README.md                   # このファイル
```
